Custom Link Relations Profile

This document defines the custom link relations used throughout the Food Manager API. Each relation includes its purpose, method, and target endpoint.

Defined Link Relations

RelationMethodTargetDescription
recipes-all GET /recipes Fetch all recipes
foods-all GET /foods Fetch all food items
categories-all GET /categories Fetch all categories
ingredients-all GET /ingredients Fetch all ingredients
add-food POST /foods Add a new food item
add-recipe POST /recipes Add a new recipe (possibly to a food item)
add-category POST /categories Add a new category
add-ingredient POST /ingredients Add a new ingredient
add-nutritional-info POST /nutritional-info Add nutritional info to a recipe
edit PUT resource-specific Edit a specific resource (food, recipe, etc.)
delete DELETE resource-specific Delete a specific resource

Usage

These relations appear inside the @controls object of hypermedia responses. They guide clients in how to navigate and interact with the API.
