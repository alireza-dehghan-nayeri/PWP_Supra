Nutritional Info Resource Profile

This resource represents the nutritional information for a specific recipe, including calories, protein, carbs, and fat content.

Fields

FieldTypeRequiredDescription
nutritional_info_idintegerNoUnique ID of the nutritional info
recipe_idintegerYesID of the associated recipe
caloriesintegerYesTotal calories
proteinfloatYesGrams of protein
carbsfloatYesGrams of carbohydrates
fatfloatYesGrams of fat

Example Representation

{
  "nutritional_info_id": 10,
  "recipe_id": 42,
  "calories": 450,
  "protein": 18.5,
  "carbs": 35.0,
  "fat": 12.0
}
