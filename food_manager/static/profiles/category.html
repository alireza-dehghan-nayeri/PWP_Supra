Category Resource Profile

This resource represents a category that can be associated with one or more recipes (e.g., "Dessert", "Main Course").

Fields

FieldTypeRequiredDescription
category_idintegerNoUnique ID of the category

  "name": "Appetizer",
  "description": "Light dishes served before the main course."
