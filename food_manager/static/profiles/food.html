Food Resource Profile

This resource represents a food item that may be associated with one or more recipes.

Fields

FieldTypeRequiredDescription
food_idintegerNoUnique identifier of the food item
namestringYesName of the food
descriptionstringNoOptional description of the food
image_urlstringNoOptional image URL
recipesarrayNoList of associated recipes (short form)

Example Representation

{
  "food_id": 123,
  "name": "Banana",
  "description": "A sweet fruit",
  "image_url": "https://example.com/images/banana.jpg",
  "recipes": [
    { "recipe_id": 1, "instruction": "Peel and eat." }
  ]
}

Link Relations

RelationMethodDescription
selfGETLink to this food item
editPUTEdit this food item
deleteDELETEDelete this food item
collectionGETGet all foods
add-recipePOSTAdd a recipe to this food (if none exists)
